# SMS Spam Detection

This project uses machine learning to classify SMS messages as **Spam** or **Not Spam**.

## Files:
- **sms_spam_detection.py**: Python script that contains the model training and prediction code.
- sms_spam.csv**: Dataset used to train the model (includes labeled spam and not spam messages).
- **model.pkl**: The trained machine learning model used for SMS classification.
- **vectorizer.pkl**: The vectorizer used for text data preprocessing.
- **requirements.txt**: List of dependencies required to run the script.
